# Changelog — sin-grill-me

All notable changes documented here.

## [0.1.0] — 2026-06-05

### Added
- Initial skill release
- SKILL.md with full spec (mindset, 4-step process, anti-patterns)
- One-question-at-a-time grilling contract
- Recommended-answer pattern (forces user reaction)
- Codebase-exploration-first rule
- Decision tree mapping (success, failure modes, assumptions, dependencies, edge cases, scope)
- Synthesis step with handoff to gsd-discuss-phase / sin-doc-coauthoring / sin-goal-mode
- Anti-patterns: multiple questions, vague answers, stopping early
- Companion tools: gsd-discuss-phase, sin-doc-coauthoring, sin-goal-mode, ceo-audit, sin-context-bridge
- Attribution to upstream Joanium/Skills (GrillMe pattern)

### Why
- gsd-discuss-phase is user-driven (user picks gray areas)
- ceo-audit is technical (security, performance, code quality)
- grill-me is design/architecture adversarial (fills the gap)
- Forms complete review loop: grill-me → gsd → ceo-audit
